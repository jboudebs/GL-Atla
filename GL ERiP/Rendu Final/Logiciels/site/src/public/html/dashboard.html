Bienvenue,

Merci de candidater à l'ENSSAT, pour poursuivre votre candidature vous devez déposez via l'onglet "Dépôt des fichiers" :

  • une copie de votre carte d'identité
  • une lettre de motivation
  • un CV
    •

    Une fois ces différents documents déposés, un gestionnaire s'occupera de les valider ou non. Si vous êtes admissible vous serez convoqué à un entretien.
    Les différentes informations seronts disponibles dans l'onglet "Vos entretiens".

    Vous saurez ou non si votre candidature est acceptée dans les quelques jours qui suivent l'entretien à partir l'onglet "Suivi candidature".
